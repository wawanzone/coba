FAQ Domain Gratis

Apa yang dimaksud dengan nama domain?

Sebenarnya pengertian nama domain mudah untuk dipahami. Nama domain dapat diibaratkan sebagai alamat rumah teman Anda, sedangkan hosting adalah bangunan rumahnya. Jika ingin sampai di rumah teman dengan cepat dan tepat, maka carilah berdasarkan alamatnya.

Tanpa nama domain, Anda harus menghafal alamat IP yang sayangnya sangat panjang dan susah untuk diingat. Selain itu, menampilkan alamat IP hanya akan memberi impresi yang negatif dari pengunjung untuk website Anda. Karena itulah, nama domain dibutuhkan karena mudah diingat dan dilafalkan.

Contoh:

Anda mempunyai toko bunga bernama Ray Florist dan ingin memudahkan konsumen menemukan Anda di internet dengan membuat website. Untuk membuat website, Anda harus menetapkan nama domain terlebih dahulu. Anda bisa menggunakan rayflorist.com sebagai nama domain.

Menggunakan nama domain akan memudahkan orang menemukan dan mengingat Anda, daripada harus menghafalkan deretan angka dalam alamat IP yang sangat memusingkan.

Apakah ada ekstensi nama domain gratis?

Untuk mendapatkan free domain, baik .COM maupun top level domain lainnya, Anda cukup berlangganan layanan web hosting murah IDwebhost. Dengan berlangganan paket web hosting IDwebhost minimal dua tahun, Anda tak perlu membeli domain berbayar secara terpisah.

Tak hanya mendapatkan domain, dengan berlangganan hosting murah IDwebhost, Anda juga akan mendapatkan web builder untuk membangun website agar lebih maksimal.

Adakah jangka waktu saya bisa menggunakan domain gratis tersebut?

Tentu saja ada. Dengan membeli web hosting murah terbaik di IDwebhost minimal paket Awesome dan Fantastic, domain Anda secara otomatis akan diperpanjang setiap tahun, selama masa kontrak hosting masih aktif.
