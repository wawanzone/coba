Paling Sering Ditanyakan (FAQ)

VPS AWS Lightsail adalah layanan Virtual Private Server (VPS) untuk membangun dan menghosting aplikasi di cloud server milik Amazon Web Services. VPS AWS Lightsail memudahkan kamu untuk membangun dan meng-online-kan website, aplikasi website dan project dengan mudah dan cepat tanpa gangguan dari pengguna lain.

Virtual Private Server (VPS) adalah layanan web hosting yang menggunakan teknologi virtualisasi dan menyediakan resource untuk pribadi. Layanan VPS ini layaknya server sendiri, tapi dengan biaya yang sangat terjangkau.

Kamu mendapatkan hak full-access untuk mengontrol dan mengelola resource VPS, bebas melakukan kustomisasi dan konfigurasi kapanpun kamu inginkan. Kamu akan mendapatkan ruang lingkup dan jangkauan server yang luas secara global secara aman.

Pastikan OS dan pilihan paket VPS sudah sesuai dengan yang kamu butuhkan agar tidak terjadi kesalahan order. Lihat TOS.

VPS AWS Lightsail kamu akan langsung aktif setelah pembayaran berhasil dikonfirmasi.

IDwebhost memastikan VPS kamu up and running. Terkait aplikasi yang kamu install dan jalankan di dalam VPS tersebut menjadi tanggungjawab pelanggan.

Kamu bisa melakukan upgrade VPS AWS Lightsail untuk menambah resource VPS yang kamu butuhkan. Apabila ingin downgrade, kamu dapat membeli VPS AWS Lightsail baru sesuai dengan spesifikasi dan konfigurasi yang akan digunakan, kemudian lakukan migrasi data dari VPS lama ke VPS baru.

VPS AWS Lightsail tidak bisa dilakukan reinstall, yang bisa dilakukan adalah membuat instance baru (termasuk mendapatkan alamat IP baru) dan menghapus instance lama. Sebelum menghapus instance lama, lakukan backup terlebih dahulu.

Kamu bisa login melalui SSH client favoritmu menggunakan SSH Key yang tersedia di member area pada menu "Connection Details".

Lightsail menawarkan berbagai sistem operasi dan template aplikasi yang diinstal secara otomatis saat kamu membuat server Lightsail baru. Template aplikasi termasuk WordPress, Drupal, Joomla!, Ghost, Magento, Redmine, LAMP, NGINX, MEAN, Node.js, Django, dan banyak lagi. Kamu dapat menginstal aplikasi tambahan dengan menggunakan SSH dalam browser atau client SSH kamu sendiri.

Image yang bisa kamu gunakan hanya yang disediakan oleh VPS AWS Lightsail.

Kamu harus melakukan backup VPS AWS Lightsail sendiri agar memiliki data cadangan.

Server AWS Lightsail kamu secara otomatis mendapatkan alamat IP pribadi dan alamat IP publik. IP pribadi dapat kamu gunakan untuk mengirimkan data dalam satu server VPS secara pribadi, gratis. IP publik dapat kamu gunakan untuk terhubung ke website atau aplikasi web dari internet, seperti melalui nama domain terdaftar atau melalui koneksi SSH atau RDP dari komputer lokal.

Lokasi data center AWS Lightsail ada 14 titik di seluruh dunia, meliputi US, Europe, dan Asia Pacific.
