FAQ

Apakah hanya bisa untuk WordPress saja?

Tidak, kamu bisa install apapun di hosting WordPress.

Apa bedanya hosting WordPress dengan hosting lainnya?

Server hosting WordPress sudah dirancang khusus untuk kebutuhan WordPress, sehingga memastikan kecepatan akses yang luar biasa cepat dan stabil untuk website WordPress.

Apakah benar space nya unlimited?

Ya. Pembatasan bukan pada space nya, tapi pada resource yang digunakan, misalnya CPU dan RAM. Namun kamu tetap diwajibkan untuk menggunakan space secara fair dan adil (fair usage) demi kenyamanan bersama. IDwebhost membatasi jumlah inodes pada akun wordpress hosting sesuai dengan paket yang digunakan. Inode adalah jumlah file yang diupload. Misalkan batasan inode nya adalah 75.000, maka Anda hanya diizinkan untuk upload sebanyak 75.000 file saja.

Apa itu fitur gratis konsultasi SEO?

Kamu dapat bergabung kedalam grup eksklusif konsultasi SEO dari IDwebhost. Group akan dipandu langsung oleh expert SEO untuk membantu website tampil di halaman pertama pencarian Google. Group eksklusif ini hanya dapat diikuti untuk yang berlangganan paket WordPress Alto.

Apakah itu fitur gratis SEO Site Audit?

Saat website mu sudah live dan selesai diisi kontennya, kamu bisa request untuk fasilitas SEO Site Audit. Tim ahli SEO kami akan meng-audit website mu dan akan memberikan berbagai rekomendasi untuk memperbaiki performa web mu di hasil pencarian Google. Fitur ini hanya tersedia untuk pelanggan paket WordPress Mezzo dan WordPress Soprano.

Apakah bandwidth saya dibatasi?

IIDwebhost tidak menetapkan batasan khusus pada resource bandwidth pada akun shared wordpress hosting yang kamu gunakan. Kami menetapkan kebijakan Fair Usage Policy (FUP) yang berlaku untuk semua akun. Artinya, kamu dapat memanfaatkan resource yang ada untuk website kamu secara wajar, selama tidak mengganggu akun hosting lain yang terletak dalam server yang sama.
