Open Source

Laravel adalah framework PHP yang bersifat open source. Selain itu hosting laravel mudah digunakan untuk membuat website dan aplikasi yang kompleks.

Security

Framework PHP laravel ini sangat mudah dipahami dan lengkap dokumentasinya. Memudahkan development saat mengelola pembuatan website.

Template Engine

FAQ

Apakah hanya bisa untuk Laravel saja?

Tidak, kamu bisa install apapun di Hosting Laravel.

Apa bedanya Hosting Laravel dengan hosting lainnya?

Server hosting Laravel sudah dirancang khusus untuk kebutuhan Laravel, sehingga memastikan kecepatan akses yang luar biasa cepat dan stabil untuk website Laravel.

Apakah Hosting Laravel mendapat fasilitas SSH?

Ya. Kamu akan mendapatkan layanan SSH untuk Hosting Laravel kamu.

Apakah saya bisa memindahkan website Laravel saya ke Hosting Laravel di IDwebhost?

Ya. Kami akan membantu melakukan proses migrasi website Laravel kamu ke IDwebhost. Selama proses migrasi berlangsung, website Laravel kamu tetap akan bisa diakses sehingga pengunjung website kamu tidak tahu bahwa website kamu sudah berpindah hosting.

Apakah benar space nya unlimited?

Ya. Pembatasan bukan pada space nya, tapi pada resource yang digunakan, misalnya CPU dan RAM. Namun kamu tetap diwajibkan untuk menggunakan space secara fair dan adil (fair usage) demi kenyamanan bersama. IDwebhost membatasi jumlah inodes pada akun wordpress hosting sesuai dengan paket yang digunakan. Inode adalah jumlah file yang diupload. Misalkan batasan inode nya adalah 75.000, maka Anda hanya diizinkan untuk upload sebanyak 75.000 file saja.

Apakah bandwidth saya dibatasi?

IIDwebhost tidak menetapkan batasan khusus pada resource bandwidth pada akun shared wordpress hosting yang kamu gunakan. Kami menetapkan kebijakan Fair Usage Policy (FUP) yang berlaku untuk semua akun. Artinya, kamu dapat memanfaatkan resource yang ada untuk website kamu secara wajar, selama tidak mengganggu akun hosting lain yang terletak dalam server yang sama.
