Apa Keunggulan Menggunakan AWS WorkMail

Mudah Kelola Email

Kamu tidak perlu menginstal software dan hardware tambahan. AWS WorkMail menangani semua patch, cadangan, dan upgrade secara otomatis. AWS WorkMail juga memudahkan kamu untuk mengelola database dan aktivitas mailing perusahaan.

Jaminan Keamanan Tinggi

Keamanan data email perusahaan kamu keamanannya terjamin. AWS WorkMail secara otomatis mengenkripsi semua data dengan kunci enkripsi yang kamu kendalikan menggunakan AWS Key Management Service (KMS). AWS WorkMail juga memberikan kontrol penuh terhadap pelokalan data.

Kompatibel Outlook

Kamu dapat terus menggunakan mail client tanpa perlu menginstal perangkat lunak tambahan, seperti plug-in untuk Microsoft Outlook . AWS WorkMail support untuk Microsoft Outlook di Windows dan Mac OS X.

Akses Dimana Saja

Kamu dapat menyinkronkan mailbox dengan perangkat iOS, Android , Amazon Fire, dan Windows Phone. Jika kamu melakukan migrasi dari server Microsoft Exchange, perangkat seluler kamu secara otomatis terhubung ke AWS WorkMail tanpa memerlukan konfigurasi ulang. Web client dengan fitur lengkap tersedia untuk mengakses email, kalender, dan kontak..

Integrasi Active Directory

Kamu akan mudah untuk mengelola pengguna dan grup dengan tool sistem manajemen yang mudah dipahami, seperti Active Directory Users and Computers. AWS WorkMail terintegrasi aman dengan Microsoft Active Directory yang sudah ada sehingga kamu dapat mengakses mailbox.

Harga Terjangkau

Kamu tidak perlu khawatir akan mengeluarkan banyak biaya agar bisa menggunakan semua fitur lengkap beserta keamanan yang tinggi dari AWS WorkMail. Semua bisa kamu dapatkan dengan harga yang terjangkau.

Terintegrasi AWS CloudTrail

Kamu akan terbebas dari risiko keamanan karena AWS WorkMail terintegrasi dengan AWS CloudTrail. CloudTrail adalah layanan log audit segala aktivitas akun AWS kamu yang mampu melakukan analisis keamanan, pelacakan perubahan, dan pemecahan masalah.

Mobile Device Management

Kamu dapat melakukan enkripsi, mengunci, menentukan persyaratan kata kunci, dan menghapus perangkat dari jarak jauh menggunakan AWS Management Console. AWS WorkMail memberlakukan kebijakan keamanan untuk perangkat seluler melalui protokol Microsoft Exchange ActiveSync.

Proteksi Spam dan Virus

Kamu akan terlindungi dari email berbahaya dengan fitur scan AWS WorkMail terhadap semua email masuk dan keluar dari spam , malware, dan virus.

Paling Sering Ditanyakan (FAQ)

AWS (Amazon Web Services) WorkMail adalah layanan email dan kalender bisnis yang aman dan terkelola yang support untuk desktop dan seluler. AWS WorkMail memiliki kemampuan untuk mengakses email, kontak, dan kalender dengan lancar menggunakan Microsoft Outlook, web browser, atau aplikasi email iOS dan Android. Kamu dapat mengintegrasikan AWS WorkMail dengan direktori perusahaan yang ada dan mengontrol enkripsi data dan lokasi penyimpanan data.

Kamu dapat mengakses AWS WorkMail dari Microsoft Outlook di Windows dan Mac OS X, serta pada perangkat seluler yang support Microsoft Exchange ActiveSync termasuk iPhone, iPad, Kindle Fire, Fire Phone, Android, dan Windows Phone. Selain itu, kamu juga bisa menggunakan aplikasi Apple Mail di Mac OS X atau aplikasi web untuk mengakses AWS WorkMail.

Ukuran file maksimal pada email kirim dan terima pada AWS WorkMail adalah 25MB.

Kamu dapat mengirim email maksimal 10.000 per hari. AWS WorkMail merupakanan layanan email bisnis namun tidak dimaksudkan untuk pengiriman email massal (bulk e-mail services).

Tidak, AWS WorkMail tidak menyediakan folder untuk publik.

Kamu dapat mengakses AWS WorkMail dengan aplikasi mail client yang mendukung protokol IMAP . Namun saat ini AWS WorkMail tidak menawarkan dukungan untuk akses email POP3.
