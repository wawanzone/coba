FAQ

Apa itu web hosting?

Web hosting adalah suatu layanan penyimpanan data bagi user yang berguna untuk menyimpan informasi, gambar, video dan beberapa konten lainnya yang akan diakses secara online. Web hosting disediakan oleh suatu penyedia jasa layanan hosting. Ada beberapa jenis layanan hosting yaitu Shared Hosting, Dedicated Hosting, VPS (Virtual Private Server) dan Colocation Server. Biasanya untuk website standar menggunakan Shared Hosting.

Apa perbedaan domain dan hosting?

Selain layanan website hosting, Kamu juga harus membeli domain. Namun, apa perbedaan web hosting dan nama domain? Web hosting adalah space server untuk situs Kamu, sedangkan domain merupakan alamat situs Kamu - misalnya, nama domain kami adalah idwebhost.com. Jika user ingin mengakses situs Kamu, maka mereka akan mengetikkan nama domain ke kolom address browser kemudian hosting akan memuat konten yang diminta.

Bagaimana memilih paket hosting yang tepat?

Tentukan apakah website ini sebagai database (pelayanan) atau hanya sebagai company profile. Misalkan berhubungan dengan database, kamu membutuhkan space untuk input data, SSL dan manajemen hosting. Sebaliknya jika hanya company profile SSL, Domain, dan hosting untuk data umum perusahaan yang diupload dalam website.

Apa itu sertifikat SSL?

Sertifikat SSL adalah file data sederhana yang menggunakan cryptographic key untuk menautkan identitas perusahaan dengan file yang tersimpan di webserver Kamu secara digital. Tujuannya adalah untuk menegaskan identitas publisher halaman web dan untuk mengurangi kemungkinan pengunjung mengakses area sensitif dari sebuah website tanpa adanya perlindungan keamanan. Layanan web hosting IDwebhost menyediakan secara gratis, namun jika Kamu membutuhkan perlindungan extra kamu bisa menambahkan layanan premium SSL.

Apa itu Database MySQL?

MySQL merupakan sistem manajemen database relasional open source RDBMS yang open source dan menggunakan bahasa query berstruktur (SQL) untuk mengakses, menambahkan, dan mengelola konten dari sebuah database. Sebagian besar sistem manajemen konten (seperti WordPress) menggunakan database MySQL untuk menyimpan data.
