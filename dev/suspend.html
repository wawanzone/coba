Penyebab Website Terkena Suspend

Penyebab domain dan hosting terkena suspend dapat terjadi karena beberapa alasan, di antaranya:

1

Pembayaran domain atau hosting yang tertunggak

Jika pengguna tidak membayar domain atau hosting tepat waktu, maka IDwebhost dapat menonaktifkan akun pengguna sampai pengguna membayar kembali.

2

Penggunaan domain atau hosting untuk kegiatan ilegal

IDwebhost dapat menonaktifkan akun pengguna jika terdeteksi bahwa domain atau hosting digunakan untuk kegiatan ilegal, seperti spamming, phishing, atau penyebaran malware.

Pelajari selengkapnya tentang spamming dan phising di bawah ini:

  1. Rekomendasi Plugin Anti Spam WordPress Terbaik [UPDATE]
  2. Malware Adalah: Kenali Pengertian, Cara Mencegah
  3. Cara Mengatasi Web Terkena Malware
3

Penggunaan sumber daya hosting yang tidak wajar

Jika aplikasi atau website yang dihosting menggunakan terlalu banyak sumber daya hosting, seperti bandwidth atau CPU, maka IDwebhost dapat menonaktifkan akun hosting untuk menjaga kestabilan server. Jika pengguna merasa kekurangan sumber daya, silakan untuk mengupgrade layanan IDwebhost yang sedang digunakan saat ini ke jenis layanan yang lebih tinggi.

4

Pelanggaran terhadap syarat dan ketentuan IDwebhost

IDwebhost memiliki syarat dan ketentuan yang harus diikuti oleh para pengguna domain atau hosting. Jika pengguna melanggar salah satu dari syarat dan ketentuan tersebut, maka IDwebhost selaku penyedia domain atau hosting dapat menonaktifkan akun pengguna.

Untuk menghindari domain atau hosting terkena suspend, sebaiknya pengguna membayar domain atau hosting tepat waktu, tidak menggunakan domain atau hosting untuk kegiatan ilegal, tidak menggunakan terlalu banyak sumber daya hosting, serta mematuhi syarat dan ketentuan yang diberikan oleh penyedia domain atau hosting.
