Apakah bandwidth itu?

Bandwidth adalah besar data transfer yang terpakai untuk 1 halaman website ditampilkan ke pengunjung dari webserver.

Misal satu page = 50 kilo bytes, dan halaman tesebut dikunjungi 300.000 x dalam satu hari, maka total pemakaian bandwidth page tersebut adalah 15.000.000.000 atau 15 Gb.

Virtual Private Server

Apa itu VPS?

VPS (Virtual Private Server) adalah teknologi server side tentang operating system dan software yang memungkinkan sebuah mesin dengan kapasitas besar di bagi ke beberapa virtual mesin. Tiap virtual mesin ini melayani operating system dan software secara independen dan dengan konfigurasi yang cepat. Dengan VPS Anda bisa mendapatkan hak akses penuh atas server layaknya Dedicated Server namun dengan harga yang murah.

Apa perbedaaan antara Shared Hosting dengan VPS?

Dalam shared hosting, website Anda menggunakan memory / RAM, Processor, Bandwidth (kecepatan akses) secara bersamaan dan berbagi dengan pelanggan yang lain dalam satu server. Dengan VPS, Anda mempunyai kebebasan untuk mengatur system file, disk space, system resource, bandwidth, memory, dll.

Apakah yang dimaksud dengan Memori Garansi dan Burst Memori?

Memori Garansi adalah quota memori (RAM) yang dialokasikan untuk account VPS Anda yang bisa Anda gunakan setiap saat. Burst Memori mengijinkan Anda untuk menggunakan memori melebihi Memori Garansi selama memori tersedia untuk digunakan sampai batas Burst.
